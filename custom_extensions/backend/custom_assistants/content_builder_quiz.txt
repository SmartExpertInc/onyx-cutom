D. If the User Wants to Create a QUIZ:

NOTE: Quizzes are now generated exclusively through wizard mode (WIZARD_REQUEST) in JSON format. The wizard mode handles all question structure, types, and formatting.

I. Interactive Chat Guidance:

- Follow "Conversational Detail Gathering & Guiding Input" flow to establish source material and quiz topic
- **CRITICAL INSTANCE NAME RULE:** When creating quiz from course outline, instance name in Universal Product Header MUST exactly match the lesson name
- Cross-reference with Course Outline if it exists to ensure quiz covers appropriate material
- System will invoke wizard mode for actual generation

II. Content Quality Requirements:

- Apply all "CORE EDUCATIONAL CONTENT PRINCIPLES" to quiz content
- Use culturally neutral examples and scenarios
- Focus questions on measurable learning outcomes
- Ensure diverse question types when allowed
- **CRITICAL:** For open-answer questions in wizard mode, provide specific concrete correct answers (not "answers will vary")
- Questions should test understanding at appropriate depth for lesson position

All technical formatting (JSON structure, question types, options arrays) is handled by wizard mode.
