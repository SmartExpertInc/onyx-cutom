## Изменения в registry.ts

### Цвета по умолчанию
Все новые шаблоны теперь используют `undefined` для цветов по умолчанию, что означает:
- `titleColor: undefined` - будет использоваться цвет темы
- `textColor: undefined` - будет использоваться цвет контента темы
- `backgroundColor: undefined` - будет использоваться фон темы
