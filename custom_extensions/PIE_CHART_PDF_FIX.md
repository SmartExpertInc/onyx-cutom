# Исправление проблемы с круговой диаграммой в PDF

## Проблема
В компоненте `PieChartInfographicsTemplate.tsx` при рендеринге в PDF круговая диаграмма отображалась полностью, но все сегменты находились в одной стороне круга, что выглядело неправильно.

## Причина
Проблема была в использовании CSS `conic-gradient` для создания сегментов круговой диаграммы. В PDF этот CSS-эффект не всегда корректно отображается, особенно при сложных градиентах.

## Решение
Заменил CSS `conic-gradient` на SVG-реализацию круговой диаграммы:

### Основные изменения:

1. **Добавлены типы для SVG элементов**:
```typescript
declare global {
  namespace JSX {
    interface IntrinsicElements {
      svg: React.SVGProps<SVGSVGElement>;
      defs: React.SVGProps<SVGDefsElement>;
      filter: React.SVGProps<SVGFilterElement>;
      feDropShadow: React.SVGProps<SVGFEDropShadowElement>;
      path: React.SVGProps<SVGPathElement>;
    }
  }
}
```

2. **Заменил CSS conic-gradient на SVG path элементы**:
- Каждый сегмент теперь создается как отдельный SVG path
- Используется математический расчет координат для правильного позиционирования сегментов
- Добавлены тени и обводка для лучшего визуального эффекта

3. **Обновил расчет позиций меток**:
- Изменил расчет углов с градусов на радианы для соответствия SVG логике
- Исправил позиционирование меток процентов

### Преимущества нового решения:

- ✅ **Лучшая совместимость с PDF** - SVG корректно отображается в PDF
- ✅ **Правильное распределение сегментов** - каждый сегмент занимает свою правильную позицию
- ✅ **Более точный контроль** - можно легко настроить тени, обводку и другие эффекты
- ✅ **Масштабируемость** - SVG автоматически масштабируется без потери качества

### Технические детали:

- Центр круга: (140, 140)
- Радиус сегментов: 120px
- Радиус для меток: 98px
- Начальный угол: -90° (верх круга)
- Направление: по часовой стрелке

## Файлы изменены:
- `frontend/src/components/templates/PieChartInfographicsTemplate.tsx`

## Результат
Теперь круговая диаграмма корректно отображается в PDF с правильным распределением сегментов по всему кругу.
