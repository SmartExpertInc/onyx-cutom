# Material Green PDF Design - Улучшенный дизайн PDF документов

## Обзор изменений

Данный документ описывает улучшения дизайна PDF документов с проектами, которые были внедрены для создания более современного, зеленого "материал" дизайна с улучшенной типографикой, размерами шрифтов, иконок и цветовой схемой.

## Основные улучшения

### 1. Цветовая схема Material Design Green

**Новая палитра цветов:**
- Основной зеленый: `#10b981` (emerald-500)
- Темно-зеленый: `#059669` (emerald-600)
- Светло-зеленый: `#d1fae5` (emerald-100)
- Фон: `#f8fafc` (slate-50)
- Текст: `#1e293b` (slate-800)

**Преимущества:**
- Более мягкие, не режущие глаз цвета
- Лучший контраст для читаемости
- Современный Material Design подход

### 2. Улучшенная типографика

**Шрифты:**
- Основной: Inter (300-800 весов)
- Дополнительный: Roboto
- Улучшенное сглаживание шрифтов

**Размеры шрифтов:**
- Основной текст: 14px (было 13px)
- Заголовки: 28px (было 24px)
- Названия папок: 16px (было 14px)
- Названия проектов: 14px (было 13px)
- Данные в ячейках: 14px (было 13px)

### 3. Улучшенные иконки

**Размеры иконок:**
- Папки: 22px (было 18px)
- Уроки: 16px (было 14px)
- Таймеры: 16px (было 14px)
- Номера проектов: 24px (было 20px)

**Визуальные эффекты:**
- Тени для иконок: `drop-shadow(0 1px 2px rgba(16, 185, 129, 0.2))`
- Улучшенная видимость и контрастность

### 4. Улучшенные отступы и пространство

**Отступы:**
- Контейнер: 32px (было 24px)
- Ячейки таблицы: 16px 20px (было 12px)
- Заголовки: 16px 20px (было 10px 12px)
- Мини-сводка: 20px (было 16px)

**Межстрочные интервалы:**
- Основной текст: 1.6 (было 1.5)
- Улучшенная читаемость

### 5. Улучшенные тени и эффекты

**Тени:**
- Контейнер: `0 10px 25px -5px rgba(0, 0, 0, 0.1)`
- Таблицы: `0 8px 25px -5px rgba(16, 185, 129, 0.1)`
- Зеленые тени для акцентов

**Градиенты:**
- Заголовки: `linear-gradient(135deg, #10b981 0%, #059669 50%, #047857 100%)`
- Папки: `linear-gradient(135deg, #ecfdf5 0%, #d1fae5 50%, #a7f3d0 100%)`
- Номера проектов: `linear-gradient(135deg, #10b981 0%, #059669 50%, #047857 100%)`

### 6. Улучшенные бейджи качества

**Размеры:**
- Отступы: 4px 10px (было 2px 6px)
- Шрифт: 12px (было 11px)
- Вес: 600 (было 500)

**Цвета:**
- Basic: улучшенный зеленый градиент
- Сохранены оригинальные цвета для других уровней

### 7. Текстуры и паттерны

**Добавлены:**
- Текстура "зерно" в заголовках
- Тонкие паттерны для глубины
- Улучшенная визуальная иерархия

### 8. Улучшенные графики

**Круговые диаграммы:**
- Улучшенные тени
- Более четкие границы
- Лучшая читаемость легенды

## Файлы изменений

### Основные файлы:
1. `projects_list_pdf_template.html` - основной шаблон списка проектов
2. `material_green_theme.css` - новый CSS файл с улучшенными стилями

### Ключевые изменения в шаблоне:

```css
/* Новые переменные CSS */
:root {
    --material-green-500: #10b981;
    --material-green-600: #059669;
    --material-green-700: #047857;
    --spacing-md: 16px;
    --spacing-lg: 24px;
    --spacing-xl: 32px;
    --radius-lg: 16px;
    --shadow-green-lg: 0 8px 25px -5px rgba(16, 185, 129, 0.1);
}

/* Улучшенный заголовок */
.pdf-header {
    background: linear-gradient(135deg, #10b981 0%, #059669 50%, #047857 100%);
    padding: 32px;
    border-radius: 16px 16px 0 0;
    box-shadow: 0 10px 25px -5px rgba(0, 0, 0, 0.1);
}

/* Улучшенные иконки */
.folder-icon {
    width: 22px;
    height: 22px;
    filter: drop-shadow(0 2px 4px rgba(16, 185, 129, 0.2));
}
```
