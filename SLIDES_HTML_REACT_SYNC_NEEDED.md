# Несоответствия между HTML и React компонентами слайдов

## Проблема
HTML шаблоны для генерации видео (`avatar_slide_template.html`) используют **другие размеры и позиции**, чем React компоненты во фронтенде. Это приводит к тому, что видео выглядит иначе, чем предпросмотр в редакторе.

## CourseOverview - Найденные несоответствия

### Логотип
- **HTML**: `top: 48px; left: 48px`
- **React**: `top: '30px'; left: '30px'`
- **Должно быть**: `48px` (как в HTML для видео 1920x1080)

### Номер страницы
- **HTML**: 
  - Позиция: `bottom: 48px`
  - Размер линии: `width: 32px; height: 2px`
  - Размер шрифта: `27px`
  - Отступ: `gap: 13px`
- **React**: 
  - Позиция: `bottom: '30px'`
  - Размер линии: `width: '20px'; height: '1px'`
  - Размер шрифта: `'17px'`
  - Отступ: `gap: '8px'`
- **Должно быть**: как в HTML

### Title Container
- **HTML**: `left: 80px`
- **React**: `left: '50px'`
- **Должно быть**: `80px`

### Title/Subtitle размер шрифта
- **HTML**: `font-size: 101px`
- **React**: (нужно проверить в коде)
- **Должно быть**: пропорционально масштабировано

### Image Container
- **HTML**: `bottom: -43px; height: 91%`
- **React**: нужно проверить

## Решение
Нужно синхронизировать все размеры между HTML и React, используя правильное масштабирование:
- HTML использует фиксированные размеры для 1920x1080
- React должен масштабировать эти размеры пропорционально размеру контейнера
